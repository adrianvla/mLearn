# FrequencyWords frequency data

The German rows in `languages/de.freq.json` and Spanish rows in
`languages/es.freq.json` are derived from the corresponding 2018 OpenSubtitles
lists in hermitdave/FrequencyWords at commit
`525f9b560de45753a5ea01069454e72e9aa541c6`.

- Source: https://github.com/hermitdave/FrequencyWords
- Source data: OpenSubtitles 2018
- Content license: CC BY-SA 4.0
- License: https://creativecommons.org/licenses/by-sa/4.0/

Each generated mLearn asset retains its source order, filters non-word tokens,
and contains the first 30,000 unique alphabetic entries for that language.
